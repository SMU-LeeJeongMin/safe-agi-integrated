# Dashboard
Streamlit 대시보드

## 실행
```bash
streamlit run Dashboard/main.py
```

## 주요 패널
1. DTO-1 Input Panel
2. Feature Engineering Panel
3. Model Explanation Panel
4. What-If Simulating Panel
5. MAML 개인화 Panel
6. DTO-5 Output Panel
7. InferenceResult 저장 Panel

## Dashboard에서 저장되는 결과

대시보드의 InferenceResult 저장 버튼은 아래 위치에 로컬 CSV를 생성합니다.

```text
Dashboard/outputs/inference_results_demo.csv
```
